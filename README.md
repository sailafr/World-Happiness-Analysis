# World-Happiness-Analysis
World Happiness Analysis Using Life Ladder Data

Kualitas hidup merupakan faktor utama yang membentuk kebahagiaan masyarakat. Di era modern, berbagai tantangan seperti kondisi ekonomi, keamanan, pembangunan, dan akses kesehatan memengaruhi kesejahteraan. Karena itu, pengukuran tingkat kebahagiaan penting untuk menilai efektivitas kebijakan publik serta membantu pemerintah merancang strategi yang lebih tepat dalam meningkatkan kualitas hidup warganya.
